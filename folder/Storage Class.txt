<div>Storage Class:</div><div><br></div><div>Every variable and function in C programming has two properties: type and storage class. Type refers to the data type of variable whether it is character or integer or floating-point value etc.</div><div>There are 4 types of storage class:</div><div>&nbsp; &nbsp; automatic</div><div>&nbsp; &nbsp; external</div><div>&nbsp; &nbsp; static</div><div>&nbsp; &nbsp; register</div><div><br></div><div>Automatic storage class</div><div>-----------------------</div><div>Keyword for automatic variable</div><div><br></div><div>auto</div><div><br></div><div>Variables declared inside the function body are automatic by default. These variable are also known as local variables as they are local to the function and doesn't have meaning outside that function</div><div><br></div><div>Since, variable inside a function is automatic by default, keyword auto are rarely used.</div><div><br></div><div>External storage class</div><div>----------------------</div><div>External variable can be accessed by any function. They are also known as global variables. Variables declared outside every function are external variables.</div><div><br></div><div>In case of large program, containing more than one file, if the global variable is declared in file 1 and that variable is used in file 2 then, compiler will show error. To solve this problem, keyword extern is used in file 2 to indicate that, the variable specified is global variable and declared in another file.</div><div><br></div><div>Example to demonstrate working of external variable</div><div>---------------------------------------------------</div><div><br></div><div>#include &nbsp;&nbsp;</div><div>void Check();</div><div>int a=5; &nbsp; &nbsp;</div><div>/* a is global variable because it is outside every function */</div><div>int main(){</div><div>&nbsp; &nbsp; a+=4;</div><div>&nbsp; &nbsp; Check();</div><div>&nbsp; &nbsp; return 0;</div><div>}</div><div><br></div><div>void Check(){</div><div>&nbsp; &nbsp;++a;</div><div>/* &nbsp;----- Variable a is not declared in this function but, works in any function as they are global variable ------- &nbsp;*/</div><div>&nbsp; &nbsp;printf("a=%d\n",a);</div><div>}</div><div><br></div><div>Output</div><div>------</div><div>a=10</div><div><br></div><div>Register Storage Class</div><div>----------------------</div><div>Keyword to declare register variable</div><div><br></div><div>register</div><div><br></div><div>Example of register variable</div><div>----------------------------</div><div>register int a;</div><div><br></div><div>Register variables are similar to automatic variable and exists inside that particular function only.</div><div><br></div><div>If the compiler encounters register variable, it tries to store variable in microprocessor's register rather than memory. Value stored in register are much faster than that of memory.</div><div><br></div><div>In case of larger program, variables that are used in loops and function parameters are declared register variables.</div><div><br></div><div>Since, there are limited number of register in processor and if it couldn't store the variable in register, it will automatically store it in memory.</div><div><br></div><div>Static Storage Class</div><div>--------------------</div><div><br></div><div>The value of static variable persists until the end of the program. A variable can be declared static using keyword: static. For example:</div><div><br></div><div>static int i;</div><div><br></div><div>Here, i is a static variable.</div><div><br></div><div>Example to demonstrate the static variable</div><div>------------------------------------------</div><div>#include &lt;stdio.h&gt;</div><div>void Check();</div><div>int main(){</div><div>&nbsp; &nbsp;Check();</div><div>&nbsp; &nbsp;Check();</div><div>&nbsp; &nbsp;Check();</div><div>}</div><div>void Check(){</div><div>&nbsp; &nbsp; static int c=0;</div><div>&nbsp; &nbsp; printf("%d\t",c);</div><div>&nbsp; &nbsp; c+=5;</div><div>}</div><div><br></div><div>Output</div><div><br></div><div>0 &nbsp; &nbsp; &nbsp;5 &nbsp; &nbsp; 10</div><div><br></div><div>During first function call, it will display 0. Then, during second function call, variable c will not be initialized to 0 again, as it is static variable. So, 5 is displayed in second function call and 10 in third call.</div><div><br></div><div>If variable c had been automatic variable, the output would have been:</div><div><br></div><div>0 &nbsp; &nbsp; 0 &nbsp; &nbsp; 0</div>
