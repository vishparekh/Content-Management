<div>Enumeration:</div><div><br></div><div>Enumeration type allows programmer to define their own data type . Keyword enum is used to defined enumerated data type.</div><div>enum type_name{ value1, value2,...,valueN };</div><div>Here, type_name is the name of enumerated data type or tag. And value1, value2,....,valueN are values of type type_name.</div><div>By default, value1 will be equal to 0, value2 will be 1 and so on but, the programmer can change the default value as below:</div><div><br></div><div>enum suit{</div><div>&nbsp; &nbsp; club=0;</div><div>&nbsp; &nbsp; diamonds=10;</div><div>&nbsp; &nbsp; hearts=20;</div><div>&nbsp; &nbsp; spades=3;</div><div>};</div><div><br></div><div>Declaration of enumerated variable</div><div>----------------------------------</div><div><br></div><div>Above code defines the type of the data but, no any variable is created. Variable of type enum can be created as:</div><div><br></div><div>enum boolean{</div><div>&nbsp; &nbsp; false;</div><div>&nbsp; &nbsp; true;</div><div>};</div><div>enum boolean check;</div><div><br></div><div>Here, a variable check is declared which is of type enum boolean.</div><div><br></div><div>Example of enumerated type</div><div>--------------------------</div><div><br></div><div>#include &lt;stdio.h&gt;</div><div>enum week{ sunday, monday, tuesday, wednesday, thursday, friday, saturday};</div><div>int main(){</div><div>&nbsp; &nbsp; enum week today;</div><div>&nbsp; &nbsp; today=wednesday;</div><div>&nbsp; &nbsp; printf("%d day",today+1);</div><div>&nbsp; &nbsp; return 0;</div><div>&nbsp; &nbsp;}</div><div><br></div><div>Output</div><div>------</div><div>4 day</div><div><br></div><div>You can write any program in C language without the help of enumerations but, enumerations helps in writing clear codes and simplify programming.</div>
