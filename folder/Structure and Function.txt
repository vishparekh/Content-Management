<div>Structure and Function:</div><div><br></div><div>In C, structure can be passed to functions by two methods:</div><div>&nbsp; &nbsp; Passing by value (passing actual value as argument)</div><div>&nbsp; &nbsp; Passing by reference (passing address of an argument)</div><div>Passing structure by value</div><div>--------------------------</div><div>A structure variable can be passed to the function as an argument as normal variable. If structure is passed by value, change made in structure variable in function definition does not reflect in original structure variable in calling function.</div><div><br></div><div>Write a C program to create a structure student, containing name and roll. Ask user the name and roll of a student in main function. Pass this structure to a function and display the information in that function.</div><div><br></div><div>#include &lt;stdio.h&gt;</div><div>struct student{</div><div>&nbsp; &nbsp; char name[50];</div><div>&nbsp; &nbsp; int roll;</div><div>};</div><div>void Display(struct student stu);</div><div>/* function prototype should be below to the structure declaration otherwise compiler shows error */</div><div>int main(){</div><div>&nbsp; &nbsp; struct student s1;</div><div>&nbsp; &nbsp; printf("Enter student's name: ");</div><div>&nbsp; &nbsp; scanf("%s",&amp;s1.name);</div><div>&nbsp; &nbsp; printf("Enter roll number:");</div><div>&nbsp; &nbsp; scanf("%d",&amp;s1.roll);</div><div>&nbsp; &nbsp; Display(s1); &nbsp; // passing structure variable s1 as argument</div><div>&nbsp; &nbsp; return 0;</div><div>}</div><div>void Display(struct student stu){</div><div>&nbsp; printf("Output\nName: %s",stu.name);</div><div>&nbsp; printf("\nRoll: %d",stu.roll);</div><div>}</div><div><br></div><div>Output</div><div>------</div><div>Enter student's name: Kevin Amla</div><div>Enter roll number: 149</div><div>Output</div><div>Name: Kevin Amla</div><div>Roll: 149</div><div><br></div><div>Passing structure by reference</div><div>------------------------------</div><div><br></div><div>The address location of structure variable is passed to function while passing it by reference. If structure is passed by reference, change made in structure variable in function definition reflects in original structure variable in the calling function.</div><div><br></div><div>Write a C program to add two distances(feet-inch system) entered by user. To solve this program, make a structure. Pass two structure variable (containing distance in feet and inch) to add function by reference and display the result in main function without returning it.</div><div><br></div><div>#include &lt;stdio.h&gt;</div><div>struct distance{</div><div>&nbsp; &nbsp; int feet;</div><div>&nbsp; &nbsp; float inch;</div><div>};</div><div>void Add(struct distance d1,struct distance d2, struct distance *d3);&nbsp;</div><div>int main()</div><div>{</div><div>&nbsp; &nbsp; struct distance dist1, dist2, dist3;</div><div>&nbsp; &nbsp; printf("First distance\n");</div><div>&nbsp; &nbsp; printf("Enter feet: ");</div><div>&nbsp; &nbsp; scanf("%d",&amp;dist1.feet);</div><div>&nbsp; &nbsp; printf("Enter inch: ");</div><div>&nbsp; &nbsp; scanf("%f",&amp;dist1.inch);</div><div>&nbsp; &nbsp; printf("Second distance\n");</div><div>&nbsp; &nbsp; printf("Enter feet: ");</div><div>&nbsp; &nbsp; scanf("%d",&amp;dist2.feet);</div><div>&nbsp; &nbsp; printf("Enter inch: ");</div><div>&nbsp; &nbsp; scanf("%f",&amp;dist2.inch);</div><div>&nbsp; &nbsp; Add(dist1, dist2, &amp;dist3);&nbsp;</div><div><br></div><div>/*passing structure variables dist1 and dist2 by value whereas passing structure variable dist3 by reference */</div><div>&nbsp; &nbsp; printf("\nSum of distances = %d\'-%.1f\"",dist3.feet, dist3.inch);</div><div>&nbsp; &nbsp; return 0;</div><div>}</div><div>void Add(struct distance d1,struct distance d2, struct distance *d3)&nbsp;</div><div>{</div><div>/* Adding distances d1 and d2 and storing it in d3 */</div><div>&nbsp; &nbsp; &nbsp;d3-&gt;feet=d1.feet+d2.feet;&nbsp;</div><div>&nbsp; &nbsp; &nbsp;d3-&gt;inch=d1.inch+d2.inch;</div><div>&nbsp; &nbsp; &nbsp;if (d3-&gt;inch&gt;=12) { &nbsp; &nbsp; /* if inch is greater or equal to 12, converting it to feet. */</div><div>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;d3-&gt;inch-=12;</div><div>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;++d3-&gt;feet;</div><div>&nbsp; &nbsp; }</div><div>}</div><div><br></div><div>Output</div><div>------</div><div>First distance</div><div>Enter feet: 12</div><div>Enter inch: 6.8</div><div>Second distance</div><div>Enter feet: 5</div><div>Enter inch: 7.5</div><div><br></div><div>Sum of distances = 18'-2.3"</div><div><br></div><div>Explanation</div><div>------------</div><div><br></div><div>In this program, structure variables dist1 and dist2 are passed by value (because value of dist1 and dist2 does not need to be displayed in main function) and dist3 is passed by reference ,i.e, address of dist3 (&amp;dist3) is passed as an argument. Thus, the structure pointer variable d3 points to the address of dist3. If any change is made in d3 variable, effect of it is seed in dist3 variable in main function.</div>
