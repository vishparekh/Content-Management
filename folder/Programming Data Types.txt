<div>Programming Data Types:</div><div><br></div><div>In C, variable(data) should be declared before it can be used in program.&nbsp;</div><div>Data types are the keywords, which are used for assigning a type to a variable.</div><div><br></div><div>Data types in C:</div><div><br></div><div>Fundamental Data Types</div><div>&nbsp; &nbsp; &nbsp; &nbsp; Integer types</div><div>&nbsp; &nbsp; &nbsp; &nbsp; Floating Type</div><div>&nbsp; &nbsp; &nbsp; &nbsp; Character types</div><div>Derived Data Types</div><div>&nbsp; &nbsp; &nbsp; &nbsp; Arrays</div><div>&nbsp; &nbsp; &nbsp; &nbsp; Pointers</div><div>&nbsp; &nbsp; &nbsp; &nbsp; Structures</div><div>&nbsp; &nbsp; &nbsp; &nbsp; Enumeration</div><div><br></div><div>Syntax for declaration of a variable:</div><div><br></div><div>data_type variable_name;</div><div><br></div><div>1) Integer data types</div><div>---------------------</div><div><br></div><div>Keyword int is used for declaring the variable with integer type. For example:</div><div><br></div><div>int var1;</div><div><br></div><div>Here, var1 is a variable of type integer.</div><div><br></div><div>The size of int is either 2 bytes(In older PC's) or 4 bytes.&nbsp;</div><div>If you consider an integer having size of 4 byte( equal to 32 bits), it can take 232 distinct states as:&nbsp;</div><div>-231,-231+1, ...,-2, -1, 0, 1, 2, ..., 231-2, 231-1</div><div><br></div><div>Similarly, int of 2 bytes, it can take 216 distinct states from -215 to 215-1. If you try to store larger number than 231-1, i.e,+2147483647 and smaller number than -231, i.e, -2147483648, &nbsp;program will not run correctly.</div><div><br></div><div>2) Floating types</div><div>-----------------</div><div><br></div><div>Variables of floating types can hold real values(numbers) such as: 2.34, -9.382 etc. Keywords either float or double is used for declaring floating type variable. For example:</div><div><br></div><div>float var2;</div><div>double var3;</div><div><br></div><div>Here, both var2 and var3 are floating type variables.</div><div><br></div><div>In C, floating values can be represented in exponential form as well. For example:</div><div><br></div><div>float var3=22.442e2</div><div><br></div><div>3) Character types</div><div>------------------</div><div><br></div><div>Keyword char is used for declaring the variable of character type. For example:</div><div><br></div><div>char var4='h';</div><div><br></div><div>Here, var4 is a variable of type character which is storing a character 'h'.</div><div><br></div><div>The size of char is 1 byte. The character data type consists of ASCII characters. Each character is given a specific value. For example:</div><div><br></div><div>For, 'a', value =97</div><div>For, 'b', value=98</div><div>For, 'A', value=65</div><div>For, '&amp;', value=33</div><div>For, '2', value=49</div>
