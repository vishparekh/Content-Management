<div>#include &lt;stdio.h&gt;</div><div>int main()</div><div>{</div><div>&nbsp; &nbsp;int n, sum = 0, remainder;</div><div>&nbsp;</div><div>&nbsp; &nbsp;printf("Enter an integer\n");</div><div>&nbsp; &nbsp;scanf("%d",&amp;n);</div><div>&nbsp;</div><div>&nbsp; &nbsp;while(n != 0)</div><div>&nbsp; &nbsp;{</div><div>&nbsp; &nbsp; &nbsp; remainder = n % 10;</div><div>&nbsp; &nbsp; &nbsp; sum = sum + remainder;</div><div>&nbsp; &nbsp; &nbsp; n = n / 10;</div><div>&nbsp; &nbsp;}</div><div>&nbsp;</div><div>&nbsp; &nbsp;printf("Sum of digits of entered number = %d\n",sum);</div><div>&nbsp;</div><div>&nbsp; &nbsp;return 0;</div><div>}</div><div><br></div><div>Description :</div><div><br></div><div>Here we are using modulus operator(%) to extract individual digits of number and adding them.&nbsp;</div><div><br></div><div>For example if the input is 98, sum(variable) is 0 initially</div><div>98%10 = 8 (% is modulus operator which gives us remainder when 98 is divided by 10).</div><div>sum = sum + remainder</div><div>so sum = 8 now.</div><div>98/10 = 9 because in c whenever we divide integer by another integer we get an integer.</div><div>9%10 = 9</div><div>sum = 8(previous value) + 9</div><div>sum = 17</div><div>9/10 = 0.</div><div>So finally n = 0, loop ends we get the required sum.</div><div><br></div>
