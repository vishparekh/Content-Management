<div>Dynamic Memory Allocation:</div><div><br></div><div>The exact size of array is unknown until the compile time i.e., time when a compier compiles code written in a programming language into a executable form. The size of array you have declared initially can be sometimes insufficient and sometimes more than required. Dynamic memory allocation allows a program to obtain more memory space, while running or to release space when no space is required.</div><div>Although, C language inherently does not has any technique to allocated memory dynamically, there are 4 library functions under "stdlib.h" for dynamic memory allocation.</div><div>Function <span class="Apple-tab-span" style="white-space:pre"><span style="white-space: normal;">	</span></span>Use of Function</div><div>----------------------------</div><div>malloc() <span class="Apple-tab-span" style="white-space:pre"><span style="white-space: normal;">	</span></span>Allocates requested size of bytes and returns a pointer first <span class="Apple-tab-span" style="white-space:pre"><span style="white-space: normal;">	</span></span>byte of allocated space</div><div>calloc() <span class="Apple-tab-span" style="white-space:pre"><span style="white-space: normal;">	</span></span>Allocates space for an array elements, initializes to zero and <span class="Apple-tab-span" style="white-space:pre"><span style="white-space: normal;">	</span></span>then returns a pointer to memory</div><div>free() <span class="Apple-tab-span" style="white-space:pre"><span style="white-space: normal;">	</span></span>Dellocate the previously allocated space</div><div>realloc() <span class="Apple-tab-span" style="white-space:pre"><span style="white-space: normal;">	</span></span>Change the size of previously allocated space</div><div><br></div><div><br></div><div><br></div><div>malloc()</div><div>--------</div><div>The name malloc stands for "memory allocation". The function malloc() reserves a block of memory of specified size and return a pointer of type void which can be casted into pointer of any form.</div><div><br></div><div>Syntax of malloc()</div><div>------------------</div><div>ptr=(cast-type*)malloc(byte-size)</div><div><br></div><div>Here, ptr is pointer of cast-type. The malloc() function returns a pointer to an area of memory with size of byte size. If the space is insufficient, allocation fails and returns NULL pointer.</div><div><br></div><div>ptr=(int*)malloc(100*sizeof(int));</div><div><br></div><div>This statement will allocate either 200 or 400 according to size of int 2 or 4 bytes respectively and the pointer points to the address of first byte of memory.</div><div><br></div><div>calloc()</div><div>--------</div><div><br></div><div>The name calloc stands for "contiguous allocation". The only difference between malloc() and calloc() is that, malloc() allocates single block of memory whereas calloc() allocates multiple blocks of memory each of same size and sets all bytes to zero.</div><div><br></div><div>Syntax of calloc()</div><div>------------------</div><div>ptr=(cast-type*)calloc(n,element-size);</div><div><br></div><div>This statement will allocate contiguous space in memory for an array of n elements. For example:</div><div><br></div><div>ptr=(float*)calloc(25,sizeof(float));</div><div><br></div><div>This statement allocates contiguous space in memory for an array of 25 elements each of size of float, i.e, 4 bytes.</div><div><br></div><div>free()</div><div>------</div><div>Dynamically allocated memory with either calloc() or malloc() does not get return on its own. The programmer must use free() explicitly to release space.</div><div><br></div><div>syntax of free()</div><div>----------------</div><div>free(ptr);</div><div><br></div><div>This statement cause the space in memory by pointer ptr to be deallocated.</div><div><br></div><div>Examples of calloc() and malloc()</div><div>---------------------------------</div><div><br></div><div>Write a C program to find sum of n elements entered by user. To perform this program, allocate memory dynamically using malloc() function.</div><div><br></div><div>#include &lt;stdio.h&gt;</div><div>#include &lt;stdlib.h&gt;</div><div>int main(){</div><div>&nbsp; &nbsp; int n,i,*ptr,sum=0;</div><div>&nbsp; &nbsp; printf("Enter number of elements: ");</div><div>&nbsp; &nbsp; scanf("%d",&amp;n);</div><div>&nbsp; &nbsp; ptr=(int*)malloc(n*sizeof(int)); &nbsp;//memory allocated using malloc</div><div>&nbsp; &nbsp; if(ptr==NULL) &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;&nbsp;</div><div>&nbsp; &nbsp; {</div><div>&nbsp; &nbsp; &nbsp; &nbsp; printf("Error! memory not allocated.");</div><div>&nbsp; &nbsp; &nbsp; &nbsp; exit(0);</div><div>&nbsp; &nbsp; }</div><div>&nbsp; &nbsp; printf("Enter elements of array: ");</div><div>&nbsp; &nbsp; for(i=0;i&lt;n;++i)</div><div>&nbsp; &nbsp; {</div><div>&nbsp; &nbsp; &nbsp; &nbsp; scanf("%d",ptr+i);</div><div>&nbsp; &nbsp; &nbsp; &nbsp; sum+=*(ptr+i);</div><div>&nbsp; &nbsp; }</div><div>&nbsp; &nbsp; printf("Sum=%d",sum);</div><div>&nbsp; &nbsp; free(ptr);</div><div>&nbsp; &nbsp; return 0;</div><div>}</div><div><br></div><div>Write a C program to find sum of n elements entered by user. To perform this program, allocate memory dynamically using calloc() function.</div><div><br></div><div>#include &lt;stdio.h&gt;</div><div>#include &lt;stdlib.h&gt;</div><div>int main(){</div><div>&nbsp; &nbsp; int n,i,*ptr,sum=0;</div><div>&nbsp; &nbsp; printf("Enter number of elements: ");</div><div>&nbsp; &nbsp; scanf("%d",&amp;n);</div><div>&nbsp; &nbsp; ptr=(int*)calloc(n,sizeof(int));</div><div>&nbsp; &nbsp; if(ptr==NULL)</div><div>&nbsp; &nbsp; {</div><div>&nbsp; &nbsp; &nbsp; &nbsp; printf("Error! memory not allocated.");</div><div>&nbsp; &nbsp; &nbsp; &nbsp; exit(0);</div><div>&nbsp; &nbsp; }</div><div>&nbsp; &nbsp; printf("Enter elements of array: ");</div><div>&nbsp; &nbsp; for(i=0;i&lt;n;++i)</div><div>&nbsp; &nbsp; {</div><div>&nbsp; &nbsp; &nbsp; &nbsp; scanf("%d",ptr+i);</div><div>&nbsp; &nbsp; &nbsp; &nbsp; sum+=*(ptr+i);</div><div>&nbsp; &nbsp; }</div><div>&nbsp; &nbsp; printf("Sum=%d",sum);</div><div>&nbsp; &nbsp; free(ptr);</div><div>&nbsp; &nbsp; return 0;</div><div>}</div><div><br></div><div>realloc()</div><div>---------</div><div>If the previously allocated memory is insufficient or more than sufficient. Then, you can change memory size previously allocated using realloc().</div><div><br></div><div>Syntax of realloc()</div><div>-------------------</div><div><br></div><div>ptr=realloc(ptr,newsize);</div><div><br></div><div>Here, ptr is reallocated with size of newsize.</div><div><br></div><div>#include &lt;stdio.h&gt;</div><div>#include &lt;stdlib.h&gt;</div><div>int main(){</div><div>&nbsp; &nbsp; int *ptr,i,n1,n2;</div><div>&nbsp; &nbsp; printf("Enter size of array: ");</div><div>&nbsp; &nbsp; scanf("%d",&amp;n1);</div><div>&nbsp; &nbsp; ptr=(int*)malloc(n1*sizeof(int));</div><div>&nbsp; &nbsp; printf("Address of previously allocated memory: ");</div><div>&nbsp; &nbsp; for(i=0;i&lt;n1;++i)</div><div>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;printf("%u\t",ptr+i);</div><div>&nbsp; &nbsp; printf("\nEnter new size of array: ");</div><div>&nbsp; &nbsp; scanf("%d",&amp;n2);</div><div>&nbsp; &nbsp; ptr=realloc(ptr,n2);</div><div>&nbsp; &nbsp; for(i=0;i&lt;n2;++i)</div><div>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;printf("%u\t",ptr+i);</div><div>&nbsp; &nbsp; return 0;</div><div>}</div>
