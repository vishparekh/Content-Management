<div>do...while Loops:</div><div><br></div><div>In C, do...while loop is very similar to while loop. Only difference&nbsp;</div><div><br></div><div>between these two loops is that, in while loops, test expression is&nbsp;</div><div><br></div><div>checked at first but, in do...while loop code is executed at first then the&nbsp;</div><div><br></div><div>condition is checked. So, the code are executed at least once in&nbsp;</div><div><br></div><div>do...while loops.</div><div>Syntax of do...while loops</div><div>--------------------------</div><div>do {</div><div>&nbsp; &nbsp;some code/s;</div><div>}</div><div>while (test expression);</div><div><br></div><div>At first codes inside body of do is executed. Then, the test expression is&nbsp;</div><div><br></div><div>checked. If it is true, code/s inside body of do are executed again and the&nbsp;</div><div><br></div><div>process continues until test expression becomes false(zero).</div><div><br></div><div>Example of do...while loop</div><div>--------------------------</div><div><br></div><div>Write a C program to add all the numbers entered by a user until user&nbsp;</div><div><br></div><div>enters 0.</div><div><br></div><div>/*C program to demonstrate the working of do...while statement*/</div><div>#include &lt;stdio.h&gt;</div><div>int main(){</div><div>&nbsp; &nbsp;int sum=0,num;</div><div>&nbsp; &nbsp;do &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; /* Codes inside the body of do...while loops are at least&nbsp;</div><div><br></div><div>executed once. */</div><div>&nbsp; &nbsp;{ &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;</div><div>&nbsp; &nbsp; &nbsp; &nbsp; printf("Enter a number\n");</div><div>&nbsp; &nbsp; &nbsp; &nbsp; scanf("%d",&amp;num);</div><div>&nbsp; &nbsp; &nbsp; &nbsp; sum+=num; &nbsp; &nbsp; &nbsp;</div><div>&nbsp; &nbsp;}</div><div>&nbsp; &nbsp;while(num!=0);</div><div>&nbsp; &nbsp;printf("sum=%d",sum);</div><div>return 0;</div><div>}</div><div><br></div><div>Output</div><div>------</div><div>Enter a number</div><div>3</div><div>Enter a number</div><div>-2</div><div>Enter a number</div><div>0</div><div>sum=1</div><div><br></div><div>In this C program, user is asked a number and it is added with sum.&nbsp;</div><div><br></div><div>Then, only the test condition in the do...while loop is checked. If the test&nbsp;</div><div><br></div><div>condition is true,i.e, num is not equal to 0, the body of do...while loop is&nbsp;</div><div><br></div><div>again executed until num equals to zero.</div><div><br></div>
