<div>Programming goto:</div><div><br></div><div>In C programming, goto statement is used for altering the normal sequence of program execution by transferring control to some other part of the program.</div><div>Syntax of goto statement</div><div>------------------------</div><div><span class="Apple-tab-span" style="white-space:pre"><span style="white-space: normal;">	</span></span></div><div>goto label;</div><div>.............</div><div>.............</div><div>.............</div><div>label:&nbsp;</div><div>statement;</div><div><br></div><div>In this syntax, label is an identifier. When, the control of program reaches to goto statement, the control of the program will jump to the label: and executes the code/s after it.</div><div><br></div><div>Example of goto statement</div><div>-------------------------</div><div><br></div><div>/* C program to demonstrate the working of goto statement.*/</div><div># include &lt;stdio.h&gt;</div><div>int main(){</div><div>&nbsp; &nbsp;float num,average,sum;</div><div>&nbsp; &nbsp;int i,n;</div><div>&nbsp; &nbsp;printf("Maximum no. of inputs: ");</div><div>&nbsp; &nbsp;scanf("%d",&amp;n);</div><div>&nbsp; &nbsp;for(i=1;i&lt;=n;++i){</div><div>&nbsp; &nbsp; &nbsp; &nbsp;printf("Enter n%d: ",i);</div><div>&nbsp; &nbsp; &nbsp; &nbsp;scanf("%f",&amp;num);</div><div>&nbsp; &nbsp; &nbsp; &nbsp;if(num&lt;0.0)</div><div>&nbsp; &nbsp; &nbsp; &nbsp;goto jump; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; /* control of the program jumps to label jump */&nbsp;</div><div>&nbsp; &nbsp; &nbsp; &nbsp;sum=sum+num;</div><div>}</div><div>jump:</div><div>&nbsp; average=sum/(i-1); &nbsp; &nbsp; &nbsp;&nbsp;</div><div>&nbsp; printf("Average: %.2f",average);</div><div>&nbsp; return 0;</div><div>}</div><div><br></div><div>Output</div><div><br></div><div>Maximum no. of inputs: 4</div><div>Enter n1: 1.5</div><div>Enter n2: 12.5</div><div>Enter n3: 7.2</div><div>Enter n4: -1</div><div>Average: 7.07</div><div><br></div><div>Though goto statement is included in ANSI standard of C, use of goto statement should be reduced as much as possible in a program.</div>
