<div>Arrays Introduction:</div><div><br></div><div>In C programming, one of the frequently arising problem is to handle similar types of data. For example: If the user want to store marks of 100 students. This can be done by creating 100 variable individually but, this process is rather tedious and impracticable. These type of problem can be handled in C programming using arrays</div><div>An array is a sequence of data item of homogeneous value(same type).</div><div><br></div><div>Arrays are of two types:</div><div><br></div><div>&nbsp; &nbsp; One-dimensional arrays</div><div>&nbsp; &nbsp; Multidimensional arrays</div><div><span class="Apple-tab-span" style="white-space:pre"><span style="white-space: normal;">	</span></span></div><div>Declaration of one-dimensional array</div><div>------------------------------------</div><div><br></div><div>data_type array_name[array_size];</div><div><br></div><div>For example:</div><div>int age[5];</div><div><br></div><div>Here, the name of array is age. The size of array is 5,i.e., there are 5 items(elements) of array age. All element in an array are of the same type (int, in this case).</div><div>Array elements</div><div><br></div><div>Size of array defines the number of elements in an array. Each element of array can be accessed and used by user according to the need of program. For example:</div><div><br></div><div>int age[5];</div><div><br></div><div>Initialization of one-dimensional array</div><div>----------------------------------------</div><div><br></div><div>Arrays can be initialized at declaration time in &nbsp;this source code as:</div><div><br></div><div>int age[5]={2,4,34,3,4};</div><div><br></div><div>It is not necessary to define the size of arrays during initialization.</div><div><br></div><div>int age[]={2,4,34,3,4};</div><div><br></div><div>Example of array in C programming</div><div>---------------------------------</div><div><br></div><div>/* C program to find the sum marks of n students using arrays */</div><div><br></div><div>#include &lt;stdio.h&gt;</div><div>int main(){</div><div>&nbsp; &nbsp; &nbsp;int marks[10],i,n,sum=0;</div><div>&nbsp; &nbsp; &nbsp;printf("Enter number of students: ");</div><div>&nbsp; &nbsp; &nbsp;scanf("%d",&amp;n);</div><div>&nbsp; &nbsp; &nbsp;for(i=0;i&lt;n;++i){</div><div>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; printf("Enter marks of student%d: ",i+1);</div><div>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; scanf("%d",&amp;marks[i]);</div><div>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; sum+=marks[i];</div><div>&nbsp; &nbsp; &nbsp;}</div><div>&nbsp; &nbsp; &nbsp;printf("Sum= %d",sum);</div><div>return 0;</div><div>}</div><div><br></div><div>Output</div><div><br></div><div>Enter number of students: 3</div><div>Enter marks of student1: 12</div><div>Enter marks of student2: 31</div><div>Enter marks of student3: 2</div><div>sum=45</div><div><br></div><div>Important thing to remember in C arrays</div><div>---------------------------------------</div><div><br></div><div>Suppose, you declared the array of 10 students. For example: arr[10]. You can use array members from arr[0] to arr[9]. But, what if you want to use element arr[10], arr[13] etc. Compiler may not show error using these elements but, may cause fatal error during program execution.</div>
