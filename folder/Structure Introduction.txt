<div>Structure Introduction:</div><div><br></div><div>Structure is the collection of variables of different types under a single name for better handling. For example: You want to store the information about person about his/her name, citizenship number and salary. You can create these information separately but, better approach will be collection of these information under single name because all these information are related to person.</div><div>Structure Definition in C</div><div>-------------------------</div><div><br></div><div>Keyword struct is used for creating a structure.</div><div><br></div><div>Syntax of structure</div><div>-------------------</div><div>struct structure_name&nbsp;</div><div>{</div><div>&nbsp; &nbsp; data_type member1;</div><div>&nbsp; &nbsp; data_type member2;</div><div>&nbsp; &nbsp; .</div><div>&nbsp; &nbsp; .</div><div>&nbsp; &nbsp; data_type memeber;</div><div>};</div><div><br></div><div>We can create the structure for a person as mentioned above as:</div><div><br></div><div>struct person</div><div>{</div><div>&nbsp; &nbsp; char name[50];</div><div>&nbsp; &nbsp; int cit_no;</div><div>&nbsp; &nbsp; float salary;</div><div>};</div><div><br></div><div>This declaration above creates the derived data type struct person.</div><div><br></div><div>Structure variable declaration</div><div>------------------------------</div><div><br></div><div>When a structure is defined, it creates a user-defined type but, no storage is allocated. For the above structure of person, variable can be declared as:</div><div><br></div><div>struct person</div><div>{</div><div>&nbsp; &nbsp; char name[50];</div><div>&nbsp; &nbsp; int cit_no;</div><div>&nbsp; &nbsp; float salary;</div><div>};</div><div><br></div><div>Inside main function:</div><div>struct person p1, p2, p[20];</div><div><br></div><div>Another way of creating sturcture variable is</div><div>---------------------------------------------</div><div><br></div><div>struct person</div><div>{</div><div>&nbsp; &nbsp; char name[50];</div><div>&nbsp; &nbsp; int cit_no;</div><div>&nbsp; &nbsp; float salary;</div><div>}p1 ,p2 ,p[20];</div><div><br></div><div>In both cases, 2 variables p1, p2 and array p having 20 elements of type struct person are created.</div><div><br></div><div>Structures within structures</div><div>----------------------------</div><div><br></div><div>Structures can be nested within other structures in C programming.</div><div><br></div><div>struct complex</div><div>{</div><div>&nbsp;int imag_value;</div><div>&nbsp;float real_value;</div><div>};</div><div>struct number{</div><div>&nbsp; &nbsp;struct complex c1;</div><div>&nbsp; &nbsp;int real;</div><div>}n1,n2;</div><div><br></div>
