<div>Pointers Introduction:</div><div><br></div><div>Pointers are the powerful feature of C and (C++) programming, which differs it from other popular programming languages like: java and Visual Basic.</div><div>Pointers are used in C program to access the memory and manipulate the address.</div><div>Reference operator(&amp;)</div><div>---------------------</div><div>If var is a variable then, &amp;var is the address in memory.</div><div><br></div><div>/* Example to demonstrate use of reference operator in C programming. */</div><div>#include &lt;stdio.h&gt;</div><div>int main(){</div><div>&nbsp; int var=5;</div><div>&nbsp; printf("Value: %d\n",var);</div><div>&nbsp; printf("Address: %d",&amp;var); &nbsp;//Notice, the ampersand(&amp;) before var.</div><div>&nbsp; return 0;</div><div>}</div><div><br></div><div>Output</div><div>------</div><div>Value: 5&nbsp;</div><div>Address: 2686778</div><div><br></div><div>Note: You may obtain different value of address while using this code.</div><div><br></div><div>In above source code, value 5 is stored in the memory location 2686778. var is just the name given to that location.</div><div><br></div><div>You, have already used reference operator in C program while using scanf() function.</div><div><br></div><div>scanf("%d",&amp;var);</div><div><br></div><div>Reference operator(*) and Pointer variables</div><div>-------------------------------------------</div><div>Pointers variables are used for taking addresses as values, i.e., a variable that holds address value is called a pointer variable or simply a pointer.</div><div>Declaration of Pointer</div><div><br></div><div>Dereference operator(*) are used to identify an operator as a pointer.</div><div><br></div><div>data_type * pointer_variable_name;</div><div>int *p;</div><div><br></div><div>Above statement defines p as pointer variable of type int.</div><div><br></div><div>Example To Demonstrate Working of Pointers</div><div>------------------------------------------</div><div><br></div><div>/* Source code to demonstrate, handling of pointers in C program */</div><div>#include &lt;stdio.h&gt;</div><div>int main(){</div><div>&nbsp; &nbsp;int *pc,c;</div><div>&nbsp; &nbsp;c=22;</div><div>&nbsp; &nbsp;printf("Address of c:%d\n",&amp;c);</div><div>&nbsp; &nbsp;printf("Value of c:%d\n\n",c);</div><div>&nbsp; &nbsp;pc=&amp;c;</div><div>&nbsp; &nbsp;printf("Address of pointer pc:%d\n",pc);</div><div>&nbsp; &nbsp;printf("Content of pointer pc:%d\n\n",*pc);</div><div>&nbsp; &nbsp;c=11;</div><div>&nbsp; &nbsp;printf("Address of pointer pc:%d\n",pc);</div><div>&nbsp; &nbsp;printf("Content of pointer pc:%d\n\n",*pc);</div><div>&nbsp; &nbsp;*pc=2;</div><div>&nbsp; &nbsp;printf("Address of c:%d\n",&amp;c);</div><div>&nbsp; &nbsp;printf("Value of c:%d\n\n",c);</div><div>&nbsp; &nbsp;return 0;</div><div>}</div><div><br></div><div>Output</div><div>------</div><div>Address of c: 2686784</div><div>Value of c: 22</div><div><br></div><div>Address of pointer pc: 2686784</div><div>Content of pointer pc: 22</div><div><br></div><div>Address of pointer pc: 2686784</div><div>Content of pointer pc: 11</div><div><br></div><div>Address of c: 2686784</div><div>Value of c: 2</div><div><br></div>
