<div>What is a local block?</div><div><br></div><div>Answer :-</div><div>---------</div><div>Statements which are enclosed in left brace ({) and the right brace (}) forms a local Block.</div><div>Local Block can have any number of statements.</div><div>Branching Statements or Conditional Statements, Loop Control Statements such as &nbsp;if,else,switch,for,while forms a local block.</div><div>These Statements contain braces , so the portion of code between two braces would be considered a local block.</div><div>Variables declared in a local block have local scope i.e they can be accessed within the block only.</div>
